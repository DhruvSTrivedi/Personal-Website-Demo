Design Principles

Proximity

  • Proximity refers to the grouping of related elements closely together to form a single visual unit, enhancing clarity and readability.
  • The principle of proximity is applied in the HTML code by grouping related elements within the same `div` tags.
  • For example, the header contains the name "Dhruv Trivedi" and the username "d5trived" grouped closely together.
  • Similarly, links to different pages are arranged close together in a table layout within the `div` tag with class "links-row", creating a visual unit.

Alignment

  • Alignment is a design principle that ensures every element on the page has some visual connection with another element. This promotes unity and organization on the page.
  • In the CSS code, alignment is achieved by centering elements using the `text-align: center;` property for classes like "centered", "name", and "header".
  • Additionally, the flex layout is used for classes like "container" and "small-image", establishing alignment among the elements within these containers.

Repetition

  • Repetition strengthens a design by unifying and tying together separate parts. Consistent use of design elements adds visual interest and aids comprehension.
  • Repetition in this code is seen in the consistent use of classes like "link", "text-primary", and "centered".
  • For example, the "link" class is repeatedly used for different link elements to provide a consistent appearance.

Contrast

  • Contrast draws the viewer's eye to the page by significantly differentiating between two elements. This can be achieved through differences in color, size, typeface, line thickness, etc.
  • Contrast in this code is seen in the use of contrasting colors (classes "background-dark", "text-primary", "text-secondary") and contrast between image and text content.
  • The titles of sections like "Cars" and "Apple iPhone Versions and Prices" are highlighted with a yellow background to stand out from the rest of the content.
  • Furthermore, the CSS animation "pulse" applies a contrast effect to the name, causing it to stand out from the background.
